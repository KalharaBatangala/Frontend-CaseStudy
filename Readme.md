# Frontend Setup

### 1. Open the frontend folder

```bash
cd Frontend-CaseStudy
```

### 2. Run using Live Server or simple server

Example:

```bash
npx serve
```

or

```bash
python -m http.server 5500
```

### 3. Open in Browser

```
http://127.0.0.1:5500
```
